IBSurgeon предлагает комплексное решение по реализации надежного и быстрого шифрования баз данных СУБД Firebird 3.0

Firebird Encryption Plugin Framework


В Firebird 3.0 появилась возможность шифровать базы данных для защиты важных данных от неавторизованного доступа: только приложения c поддержкой шифрования смогут работать с зашифрованными БД.

IBSurgeon, официальный вендор-партнер Firebird и платиновый спонсор Firebird Foundation, предлагает Firebird Encryption Plugin Framework (далее FEPF) - готовый к использованию шаблон плагина шифрования, руководство по использованию и техническую поддержку. FEPF включает исходные тексты плагина (RSA+AES). Лицензия дает право на неограниченное распространение плагина с пользовательскими приложениями.

Ключевые возможности FEPF:
* Быстрое шифрование/дешифрование с очень низкой потерей производительности
* Сильные алгоритмы шифрования (RSA+AES)
* Простое встраивание в пользовательские приложения
* возможность использования стандартных инструментов (gbak, gfix)
* исходные тексты

Более полную информацию можно получить на нашем сайте - /firebird-encryption-plugin-framework/.

С любыми вопросами обращайтесь на support@ib-aid.com.
