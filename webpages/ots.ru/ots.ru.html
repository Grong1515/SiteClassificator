OST24™ is a transaction processing engine for robust, secure and cost-effective support for both financial and non-financial transactions acquired from multiple channels.

OST24 allows financial institutions, mobile operators and merchants to incorporate new technologies and delivery channels. As conversion of the core processing systems is very expensive and risky, the product ensures seamless integration with the legacy computing applications.

OST24 provides a comprehensive functionality to manage traditional transactions as well as emerging transaction types. It supports leading payment cards and offers standard interfaces to leading devices and host systems.

Open Transaction Systems promptly responds to the modern technology trends of the industry. The latest features implemented in the product include support of Verified by VISA and MasterCard Secure Code payments over the Internet, multi-application EMV Java cards, smart loyalty as well as mobile payments.

OST24 delivers the lowest up-front implementation costs on the market. Moreover, a `try and buy' option is available.
