                                                                                                                                            Приглашаем на курсы подготовки к олимпиаде по экологии

                                                                                                                                            Лицей 1561 совместно с ГБОУДО "Московский детско-юношеский центр экологии, краеведения и туризма"
                                                                                                                                            проводит подготовку к Всероссийской олимпиаде школьников по экологии для учащихся Юго-западного округа.
                                                                                                                                            Занятия по экспресс-подготовке к муниципальному этапу пройдут:
                                                                                                                                            14.10 с 16:00-18:00
                                                                                                                                            15.10 с 9:00-12:00
                                                                                                                                            19.10 с 16:00-18:00
                                                                                                                                            21.10 с 16:00-18:00

                                                                                                                                            Занятия включают повторение (освоение) основных понятий, закономерностей курса экологии. Для 7-11 классов.

                                                                                                                                            Для того, чтобы узнать детали, нажмите кнопку "подробнее".
